# PyQt5 Calculator App

A simple desktop calculator built with **PyQt5** by **Fahad Hussain**. It supports basic arithmetic operations in a neat GUI.

## 🚀 Features

- Basic operations: `+`, `−`, `×`, `/`
- Clear, Delete, and Decimal support
- Styled UI with custom buttons
- PyQt5-based desktop app

## 🖥️ Preview

> You can customize the icon path or UI colors in the source code.

## 📦 Requirements

- Python 3.x
- PyQt5
